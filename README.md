# mi-band-faces

自制小米手环表盘（小米手环10）。

## 安装

编译产物为各子目录的`output/*.face`。

[表盘自定义工具](https://www.bandbbs.cn/threads/9797/)

[AstroBox](https://astrobox.online/)
